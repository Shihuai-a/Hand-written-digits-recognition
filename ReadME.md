# **Resnet18说明文档**
这个模型是经典resnet18的经典架构，预测32x32的CIFAR10数据集。

## 一、数据集处理
利用transforms.Compose方法将对数据集的处理打包：对训练集采用RandomCrop（随机裁剪），RandomHorizontalFlip（随机反转），ToTensor（转为张量），Normalize（归一化）前面俩目的是为了数据增强，以增强数据的多样性，预防过拟合问题。最后一个归一化是为了防止梯度下降缓慢，梯度爆炸问题而做的。对验证集只需要后两个操作，因为这时候没必要进行数据增强。往后的代码就是普通的加载数据集以及分批次处理

## 二、残差连接模块(BasicBlock)
两个（卷积+BN），后连一个下采样模块，整体构成残差连接模块，这是resnet模型的核心，是其能进行深度网络训练的基础。

## 三、resnet18模型核心
初始卷积层->4个BasicBlock->池化+全连接->softmax输出概率完成分类任务
在此处通过_make_layer方法定义了残差连接中basicBlock下采样模块(downsample)，即首先判断输入通道是否等于输出通道从而选择downsample的有无
下采样模块通过nn.Sequential将一个卷积和一个BN层打包形成

## ##四、训练模型
经典的模型训练架构，进行epochs轮的训练，每一轮通过batch_size把数据分为几个批次进行训练，训练完后得到训练准确率，每一轮训练完后进行验证集的验证，得到验证准确率

## 五、模型预测
训练完模型后利用predict_image函数进行图像预测


# 结果
利用此模型训练CIFAR10数据集，得到准确率约为86%
